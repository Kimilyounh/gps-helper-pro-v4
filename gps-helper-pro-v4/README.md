# GPS Helper Pro v4.1

대리운전용 GPS 안정화 보조 앱입니다.

## v4.1 반영
- 자동점검 ON 시 Android Foreground Service 시작
- 앱을 닫아도 상태바 알림으로 GPS 자동점검 유지
- 20초 주기 백그라운드 GPS 상태 점검
- 30초 이상 위치 미갱신 시 자동 복구
- 정확도 30m 초과 시 자동 재측정
- GPS OFF/신호 약함 음성 안내 및 알림
- 하루 GPS 끊김 횟수 저장
- 부팅 후 자동 모니터링 재시작 지원
- 현재 좌표 복사, 속도(km/h), LTE/5G 상태 표시

## 주의
이 앱은 기기의 위치 상태를 보조 관리하는 앱입니다. 다른 앱이나 배차 시스템을 제어하지 않습니다.
